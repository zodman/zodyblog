Conary PackageManagement .. un manejador de paquetes para distros con poderes pythonicos.

Conary es un sistema de gestión de paquetes libre creado por rPath Inc y distribuido bajo los términos de la Common Public License Para empaquetar software tienes que programar en python. Por lo que esta charla se mostrara como una aplicacion en python puede automatizar empaquetamiento de software para linux. Asi como de como interactura en el API de conary y crear nuestra propia cliente de conary. Description: Acerca de conary: Conary enfoca en la instalación de paquetes a través de la resolución de dependencias automatizada, a diferencia del repositorio Linux distribuido en-linea, y provee un conciso y fácil de usar lenguaje de descripción basada en Python, para especificar como construir un paquete. Esto es usado por las distribuciones foresight linux, y rPath Linux. Conary actualiza sólo aquellos archivos específicos en los paquetes que tienen que ser actualizados; este comportamiento minimiza el ancho de banda y requerimientos de tiempo para actualizar paquetes de software. En resumen: conary esta echo en python y sirve para empaquetar software para linux. Donde entra python en la charla Para lograr empaquetar software tienes que programar en python el recipe(receta) de acuerdo al tipo de paquete tienes que saber que acciones vas a tomar (si el paquete eses un programa en python,perl,automake,autoconf,scons, etc etc). Seria bueno estar relacionado con procesos de compilacion o instalacion de software desde la fuente, con los tipicos: ./configure; make; make install; python setup.py build; python setup.py install Existe un software en foresight que utilizan varias distros. Packagekit este software es un fronted para varios manejadores de paquetes(apt,yum,pacman,conary) Para hacer funcionar PackageKit y conary se tuvo que programar e interactuar con conary a nivel de codigo y API. Y apartir de esto cree un modulo(conary-lib) que es un ejemplo de la implementacion del Conary API. Conferencia La conferencia se hablara de los siguiente: Pequeña introduccion a conary, que es, como funciona, y por que es un nuevo paradigma de empaquetamiento Empaquetar software para conary, se mostrara lo facil que es crear un recipe ( y este recipe esta en python) Y como dara como resultado un software empaquetado y listo para usar en foresight linux.
